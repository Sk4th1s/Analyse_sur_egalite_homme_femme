# Analyse_sur_egalite_homme_femme

Cette entreprise respecte-t-elle le principe d’égalité professionnelle entre les femmes et les hommes ?

Pour répondre à cette problématique, j'ai mis en place des indicateurs sur plusieurs thème (salaires, promotions, répartition par service, temps de travail, type de contrat et les expositions aux risques). Grace à ces indicateurs j'ai calulé un score global (basé sur le salaire, le taux d'augmentation et le top 10 des rémunérations) avec un résultat à 67%, 
j'ai constater que cette entreprise n'avait pas atteint son objectif de 85%
